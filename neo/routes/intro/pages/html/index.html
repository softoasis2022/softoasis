네오앱 (NEO App)

현실과 가상의 조화, 당신의 새로운 디지털 생활 공간.

🌐 월드 시스템

네오앱의 중심 공간. 여러 월드를 여행하고, 유저들과 실시간으로 상호작용할 수 있는 메타버스 기반 기능.

  • 실시간 접속 월드 목록
  • 월드 이동 포탈
  • 개별 월드 인터랙션
  • 3D/2D 혼합 UI 기반

💬 채팅 시스템

월드 채팅, 일방채팅 등 다양한 소통 방식 제공

  • 월드 채팅 : 같은 월드 유저와 대화
  • 일방 채팅 : 1:1 대화 가능
  • 실시간 소켓 기반 초고속 메시징
  • 안드로이드 안정화를 위한 WebSocket 강제 적용

🙋 개인 기능

내 정보와 활동을 관리하는 개인 메뉴.

  • 프로필 이미지 관리
  • 나의 활동 기록
  • 나의 아이템, 환경 설정

🙋 네오앱에서 제공하는 다양한 컨텐츠

네오앱의 월드, 채팅방, 개인 대화 안에서 유저가 즐기고 참여할 수 있는 다양한 컨텐츠를 제공합니다.

  • 🎲 **주사위·가위바위보 미니게임** — 채팅방/개인대화에서 즉석으로 즐기는 미니게임
  • 💘 **매칭·소개팅 기능** — 개인 대화 기반 자동 추천 및 실시간 매칭
  • 🏆 **월드 미션/퀘스트 시스템** — 월드 탐험 중 수행 가능한 미니 퀘스트
  • 🛒 **아이템 교환·거래 콘텐츠** — 유저 간 아이템 교환이나 월드 내 상점 시스템
  • ⭐ **호감도·친밀도 시스템** — 대화·활동에 따라 상대방과의 친밀도 상승
  • 📊 **랭킹·이벤트 시스템** — 월드 참여도/게임 결과에 따라 랭킹 제공
