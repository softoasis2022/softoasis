네오앱 (NEO App)

현실과 가상의 중간지점, 당신의 새로운 디지털 생활 공간.

🌐 월드 시스템

네오앱의 중심 공간. 여러 월드를 여행하고, 유저들과 실시간으로 상호작용할 수 있는 메타버스 기반 기능.

  • 실시간 접속 월드 목록
  • 월드 이동 포탈
  • 개별 월드 인터랙션
  • 3D/2D 혼합 UI 기반

💬 채팅 시스템

월드 채팅, 일방채팅 등 다양한 소통 방식 제공

  • 월드 채팅 : 같은 월드 유저와 대화
  • 일방 채팅 : 1:1 대화 가능
  • 실시간 소켓 기반 초고속 메시징
  • 안드로이드 안정화를 위한 WebSocket 강제 적용

🙋 개인 기능

내 정보와 활동을 관리하는 개인 메뉴.

  • 프로필 이미지 관리
  • 나의 활동 기록
  • 나의 아이템, 환경 설정

⚙ 설정 메뉴

  • 알림 설정
  • 계정 보안
  • 데이터 관리
  • 앱 테마/모드 변경
